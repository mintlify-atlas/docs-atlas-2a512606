Announcement Window+ reads your Dwarf Fortress game log in real time and surfaces announcements, combat reports, and events in a separate dual-pane window. Configure exactly which events appear in each pane, set custom colors by category, and highlight specific words — all without interrupting your fortress.

## How it works

Announcement Window+ monitors your Dwarf Fortress `annc.log` file (written by DFHack) and categorizes every line using a set of regular expression filters. Each announcement is tagged with a **group** (for color) and a **category** (for per-window visibility control). You can show combat spam in one pane while keeping important alerts in the other.

<Steps>
  <Step title="Enable DFHack logging">
    Install DFHack and configure it to write announcements to `annc.log`. This gives Announcement Window+ a live feed of game events.
  </Step>
  <Step title="Install and launch">
    Download the Windows release or run `python run.py` directly. The app opens with two announcement panes ready to use.
  </Step>
  <Step title="Connect to your fortress">
    Click **Settings → Set Directory** and point the app at your `annc.log` file. Announcements start flowing immediately.
  </Step>
  <Step title="Customize filters and colors">
    Right-click any window to toggle tags, change fonts, or open the Filter Configuration dialog to control exactly what appears in each pane.
  </Step>
</Steps>
