Announcement Window+ is a Python application that interfaces with Dwarf Fortress to print announcements and combat reports to a dedicated window — separate from the game itself. No more pausing the game to check what just happened.

## What it does

Dwarf Fortress surfaces a constant stream of game events: sieges, moods, creature sightings, combat blow-by-blow, and more. By default, reviewing these requires interrupting gameplay. Announcement Window+ monitors the game log in real time and routes events into a purpose-built display you can position anywhere on your screen.

The application polls `annc.log` (generated by DFHack) once per second, classifies each line using regular-expression filters, and renders it with configurable color coding and font settings.

## Dual-pane layout

Announcement Window+ opens two independent text panes, stacked vertically and separated by a draggable divider. Each pane has its own filter configuration, so you can dedicate one pane to a specific category of events while the other shows something entirely different.

For example:

- **Window 1** — standard announcements (tag `a`): visitors, seasons, mandates, moods
- **Window 2** — combat reports (tag `r`): a live feed of every strike, block, and dodge during a siege

Each pane scrolls independently, supports its own font, and can be cleared without affecting the other.

## Python compatibility

Announcement Window+ was originally written for Python 2.7.10 and now supports Python 3. It should work with any Python version greater than 2.3. On Windows, a standalone `.exe` release is available that requires no Python installation at all.

## DFHack requirement

Announcement Window+ reads from `annc.log`, a file that Dwarf Fortress does not produce on its own. [DFHack](https://dfhack.org/) provides the `annc-monitor` script that intercepts announcement events and writes them to disk. DFHack must be installed and configured before Announcement Window+ can receive any data.
